# In-Game Editor

The mod comes with an in-game route editor to:
- edit existing routes in-game with a nice interface
- adding new routes to the mod
- preview the routes

## Usage

| ------- | ------------ |
|open route editor| `R-Ctrl` |
|move a marker | `L-Ctrl` |
|delete a marker | `Del` |
|place a marker  | `L-Ctrl` |
|add a new marker | `>` |
